# Hướng dẫn Khởi chạy và Truy cập Odoo 17

## Đường dẫn Truy cập (Localhost)
Hệ thống Odoo Web Demo của bạn hiện đang được chạy nội bộ trên máy tính. Để sử dụng, hãy mở trình duyệt web và truy xuất vào đường dẫn sau:

👉 **[http://localhost:8069](http://localhost:8069)**

- **Tài khoản (Email):** `admin`
- **Mật khẩu:** `admin`

### Cách Khởi động lại Server (khi tắt máy tính)
Hệ thống Odoo và Cơ sở dữ liệu sẽ bị tắt nếu bạn khởi động lại máy tính. Để bật lại, hãy thực hiện các bước sau:
1. Mở thư mục gốc chứa các cài đặt (`c:\Users\haptn\Downloads\Ha.Odoo`).
2. Nhấn đúp chuột vào tệp script: **`start_odoo.bat`**.
3. Chờ cửa sổ màn hình đen (Console) chạy khoảng 5 giây.
4. Truy cập lại đường link phía trên bằng trình duyệt của bạn.
